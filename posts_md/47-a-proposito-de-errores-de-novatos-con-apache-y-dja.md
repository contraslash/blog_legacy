# A propósito de errores de novatos con Apache y DjangoAntes de en serio comenzar con este menaje, no olviden instalar la librería de apache mod-wsgi

`sudo apt-get install libapache2-mod-wsgi`

y activarla

`sudo a2enmod wsgi`
